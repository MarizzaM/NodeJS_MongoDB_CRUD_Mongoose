# NodeJS_MongoDB_CRUD_Mongoose
CRUD REST API using NodeJS | Express | Mongoose

Terminal

    npm init
    
    npm install express
    
    npm install mongodb
    
    npm install mongooose
    
    npm install nodemon --save-dev
    
package.json

        "main": "app.js",
        "scripts": {
            "start": "nodeman app.js"
        }
      
Terminal

      npm install -g nodemon --save-dev  
      
      nodemon run start
